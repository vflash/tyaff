## 🧠 Что я помню

### О проекте Tyaff
- **Tyaff** — это легковесная альтернатива React для создания UI
- Ключевые особенности: нет Virtual DOM, прямая работа с DOM, мемоизация, контексты
- Проект на русском, витрины на английском (для Hacker News и международной аудитории)

### О языковой политике
- **Разработка ведётся на русском** — это источник правды
- **Витрины на английском** — README.md, DOCS.md для международной аудитории
- **Структура файлов:**
  - `README.ru.md` — русская витрина (оригинал)
  - `README.md` — английская витрина (перевод)
  - `DOCS.ru.md` — русская документация (оригинал)
  - `DOCS.md` — английская документация (перевод)
  - Все остальные файлы (SPEC.md, DEVNOTES.md, AGENTS.md, AI*.md, INBOX.md) — только на русском

### О структуре проекта
- **Корень проекта** содержит все MD файлы (пока не реорганизовано)
